# Student-Placement-Predictor
A Machine Learning project using Logistic Regression to predict student placement based on CGPA and IQ


## 📊 Dataset Overview
The dataset contains information about students including:
* **CGPA**: Cumulative Grade Point Average
* **IQ**: Intelligence Quotient
* **Placement**: Target variable (1 = Placed, 0 = Not Placed)

## 🛠️ Tech Stack Used
* **Python**
* **Pandas & NumPy** (Data Cleaning & Manipulation)
* **Scikit-Learn** (Machine Learning Model)
* **Logistic Regression** (Algorithm used for Classification)

## 📈 Model Performance
* Achieved an accuracy score of **80% (0.8)** on the test dataset using Logistic Regression after scaling the features with `StandardScaler`.

## ⚙️ How to Run
1. Open Google Colab or Jupyter Notebook.
2. Upload the `placement.csv` dataset.
3. Run all cells sequentially.
